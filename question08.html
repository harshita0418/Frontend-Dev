Understanding Content, Padding, Border, and Margin

Default Box Model

Margin (Orange)
Border (Blue)
Padding (Green)

Content Area

Width: 300px

In the default box model, width and height only apply to the content area. Padding and border are added to the total size.

Box-Sizing: Border-Box

Margin (Orange)
Border (Blue)
Padding (Green)

Content Area

Width: 300px (total)

With box-sizing: border-box, width includes content, padding, and border. This makes sizing more predictable.
