Understanding content-box vs border-box

content-box (default)

Width: 300px

Padding: 20px

Border: 5px

Total width: 350px

With box-sizing: content-box (default), padding and border are added to the specified width. Total width = width + padding + border.

border-box

Width: 300px

Padding: 20px

Border: 5px

Total width: 300px

With box-sizing: border-box, padding and border are included within the specified width. Total width = width (includes padding and border).
