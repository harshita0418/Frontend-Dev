Responsive CSS Cheat Sheet

Filters

Apply visual effects to images. Example: filter: blur(8px);

Media Queries

Adapt styles to screen sizes. Example: @media (max-width: 600px) { }

Flexbox

Create flexible layouts. Example: display: flex; justify-content: center;

CSS Grid

Build complex layouts. Example: display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Responsive Typography

Scale fonts fluidly. Example: font-size: clamp(1rem, 2vw, 1.5rem);

Breakpoints

Mobile: <600px, Tablet: 600-900px, Desktop: >900px
