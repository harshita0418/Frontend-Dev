Responsive Design Explained

Responsive web design is an approach to web development that makes web pages render well on a variety of devices and window sizes. It involves using flexible layouts, flexible images, and media queries to adapt content to different screen sizes.

Quick Reference

Flexbox

Display: flex; gap: 1rem;

Grid

Display: grid; grid-template-columns: repeat(3, 1fr);

Clamp

Font-size: clamp(1rem, 2vw, 1.5rem);
