Gradient Background Demonstration

Linear Gradient Background

This section uses a linear gradient that transitions from one color to another in a straight line. The gradient goes from purple to blue.

CSS: background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

Radial Gradient Background with Parallax

This section uses a radial gradient that radiates from a center point. The gradient creates a circular color transition effect.

CSS: background: radial-gradient(circle, #f093fb 0%, #f5576c 100%);

This section also has background-attachment: fixed for a parallax-like effect when scrolling.
